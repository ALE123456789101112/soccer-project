Input Phase:
Users input each team’s name along with its corresponding win percentage.

League Setup:
Users can add as many teams as they wish to their league.

Match Simulation:
Each team competes against every other team twice: once at their home venue and once away.
The win percentage is used to simulate the outcome of each game.
Home Advantage Adjustment:
Home teams are given an advantage.
The away team’s win percentage is reduced by a random value between 1% and 10%.

Outcome Determination:
Teams’ win percentages are compared to determine the match result:
If equal, the match is a draw, and each team earns 1 point.
If one team’s percentage is higher, that team wins and receives 3 points.
The team with the lower percentage loses and receives no points.

League Conclusion:
After simulating all matches, a league table is generated.
The table displays each team’s total points accumulated throughout the league.
